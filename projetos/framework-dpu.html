Framework DPU

A Defensoria Pública da União é instituição essencial à função jurisdicional do Estado, incumbindo-lhe a orientação jurídica e a defesa dos necessitados, em todos os graus, perante o Poder Judiciário da União.

Sua missão não é desenvolver sistemas, contudo suas divisões precisam de suporte em informática e necessidade de customizar softwares doados por outros órgãos. Logo, uma equipe foi criada para suprir essa carência e conforme a demanda aumentava foi necessário customizar um Framework para agilizar o desenvolvimento de novas demandas.

O Framework contemplava recursos importantes os quais todos os sistemas futuros teriam. Como por exemplo, segurança de somente registrar usuários autorizados que estivessem acesso a rede DPU. Com isso, o usuário que fosse desligado do órgão perderia automaticamente o acesso aos sistemas, sem a necessidade de desvincula-los manualmente para cada sistema.

Destaca-se, que o front-end foi customizado para acessível para ambientes desktops e mobile. Tendo somente arquivos CSS e JS que seria realmente utilizado. Dessa forma, foi possível criar sistemas leves, compactos, mas ao mesmo tempo eficientes e organizados em uma arquitetura MVC.

Enfim, esse framework foi a base de todos os outros sistemas posteriores e atualização dos existentes para essa padronização, evitando a duplicidade de codificação além de criar uma identidade visual coesa e sólida.
