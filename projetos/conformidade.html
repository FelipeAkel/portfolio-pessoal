Conformidade

Inicialmente o sistema irá executar todo mês a leitura de um arquivo xml, excel, dos dados contidos nesse documento e registra-los em um banco de dados. Além disso, o sistema irá apresentar os painéis, dashboard, gráficos dos uploads do dados realizados até o presente momento, para análise das informações e ajuda na tomadas de decisões. O Conformidade contém um ciclo que se renova a cada ano o qual sua atualizados os dados dos dashboards para o ano atual.

Destaca-se, que os dados registrados serão comparados com outra fonte de dados. Dessa forma, será possível encontrar inconsistências, perdas de informações e processo ajudando o administrador a identificar e tomar medidas para solucionar o problema.
