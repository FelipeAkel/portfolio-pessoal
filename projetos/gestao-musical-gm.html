Sistema Gestão Musical - GM

O sistema Gestão Musical - GM trata-se de um website responsável por apresentar o porfólio de um artista da música.

Destaca-se, que o sistema é composto de 2 layouts distintos responsáveis por montar os ambientes. Primeiro, trata-se da área dos internautas de livre acesso. Segundo, trata-se da área dos administradores de acesso restrito. Dessa forma, será possível gerenciar as informações apresentadas do website além de garantir a segurança das informações.


Tecnologias utilizadas

Laravel

O Sistema GM foi desenvolvido em Laravel.

Sistema de Gerenciameto de Bando de Dados - SGBD

O Banco de Dados utilizado foi o MySQL Workbench. O Sistema GM é composto de um Banco de Dados Relacional.

Tema área do internauta: MxTonz

MxTonz é o template utilizado na Área dos Internautas baseado em Bootstrap 4.x

Tema área administrativa: Metronic

Metronic é o template utilizado na Área Administrativa baseado em Bootstrap 4.x
